# Code of Conduct

## Our pledge

We pledge to make participation in Morris PeopleSoft Utilities a harassment-free experience for everyone.

## Our standards

Examples of positive behavior:

- Respectful, constructive feedback
- Focusing on what is best for the community and college BA users
- Accepting responsibility and apologizing for mistakes

Unacceptable behavior includes harassment, trolling, or publishing others’ private information.

## Enforcement

Report concerns via GitHub Issues (private contact if needed through the maintainer [hackmods](https://github.com/hackmods)).
