## Detailed description

Morris PeopleSoft Utilities (MPU) helps college business analysts and technical staff work faster in Oracle PeopleSoft—on both Classic and Fluid pages.

**For business analysts**
• Environment labels so you always know which system you’re in  
• Favorites with categories and CSV import/export  
• Page Information without disruptive CTRL+J  
• Helpers for Advanced Search and Correct History  

**For technical staff**
• One-click PeopleCode/SQL trace using your existing PeopleSoft access  
• Field Inspector to reveal RECORD.FIELD identities  
• New Window / deep-link workflows  

**Privacy & compliance**
• Settings stored only on your device (`chrome.storage.local`)  
• No password storage and no telemetry  
• Optional host allowlist if your organization wants tighter scope  

**Accessibility**
• Designed toward AODA / WCAG 2.1 AA for extension UI
