# ADR 0007: GitHub product home and release pipeline

## Status

Accepted

## Context

`hackmods/Morris-Peoplesoft-Utilities` should be the public product home for docs, issues, and release zips used for Store upload.

## Decision

Ship modern README, wiki sources, community health files, CI on PR/main, and tag-driven release workflow that publishes the Store zip. Credit hackmods and upstream PS Utilities authors.

## Consequences

Clear support channel; reproducible artifacts; Store uploads come from GitHub Releases.
