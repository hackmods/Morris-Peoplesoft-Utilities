# ADR 0005: AODA via WCAG 2.1 AA

## Status

Accepted

## Context

College users in Ontario contexts need accessible tooling. Extension chrome must not introduce avoidable barriers.

## Decision

Target WCAG 2.1 Level AA for popup, options, and injected bar/dialogs. Enforce with axe CI and manual keyboard checklists.

## Consequences

Slightly more UI structure; better keyboard/SR support; documented known limits on Oracle pages.
