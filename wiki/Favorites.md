# Favorites (Shortcuts)

Favorites store Servlet, Menu, Component, Market, Parameters, Category, SubCategory, Description, and optional Notes.

## Bar — Shortcuts menu

- Open **Shortcuts** on the utilities bar for a hierarchical menu (Category → SubCategory → items), matching classic PS Utilities.
- **Add to Shortcuts** opens the Shortcut Details dialog (Description*, Category and SubCategory via select-or-type, optional Parameters and Notes).
- Open a shortcut in the same window, or use the new-window control on each item (when New Window is enabled).
- **Recent** components remain available beside the menu when present.

## Pages menu

- **Pages** lists all delivered component tabs/pages (helps when horizontal tabs overflow).
- The current page is highlighted; other entries activate the delivered PeopleSoft tab control.

## Options / import

- CSV import/export from Options (columns documented in the Options UI)
- JSON backup includes favorites plus other settings

**Caution:** Parameters and notes may contain business keys. Protect exported files.
