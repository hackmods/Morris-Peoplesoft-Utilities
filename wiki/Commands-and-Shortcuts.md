# Commands and shortcuts

| Action | How |
|---|---|
| Open popup toggles | Click extension icon |
| Open Options | Popup → Options, or right-click extension → Options |
| Page Info | Bar → Page Info |
| Toggle Field Inspector | Bar → Inspect; **Escape** exits |
| Toggle Trace | Bar → Trace ON/OFF |
| Add Favorite | Bar → ★ |
| Close dialogs | **Escape** or Close button |

PeopleSoft itself defines many shortcuts; MPU avoids claiming global accelerators in v1 to reduce conflicts. Prefer bar buttons.
