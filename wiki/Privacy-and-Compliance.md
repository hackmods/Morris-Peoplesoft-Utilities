# Privacy and compliance

- Local settings only; no telemetry; no passwords
- Side Panel uses `sidePanel` + `tabs` only to open Favorites into the active PeopleSoft tab and refresh Page Info — never reads form values
- Login bar placement looks for a password field only to choose mount position — never reads credentials
- Full policy: repository `docs/privacy.md`
- Optional host allowlist: `docs/compliance-host-allowlist.md` (default off for public Store parity)
