# Use cases — Technical

- **Trace:** Configure PeopleCode/SQL flags in Options → Tracing, then toggle Trace on the bar. Requires security access to delivered trace components. Cross-tab sync updates other tabs.
- **Field Inspector:** Orange icons beside fields (Classic parity); hover shows name on the bar; click locks; Escape exits.
- **New Window:** Open the current component in a `_newwin` style window/tab.
- **Backup:** Export full settings JSON from Favorites tab tools.
