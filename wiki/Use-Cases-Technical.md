# Use cases — Technical

- **Trace:** Configure PeopleCode/SQL flags in Options → Tracing, then toggle Trace on the bar. Requires security access to delivered trace components. Cross-tab sync updates other tabs. 🔒 means UTILITIES access is missing.
- **Field Inspector:** Orange icons beside fields (Classic parity); Rec/Fld/Row (+ Label / Work badge); click locks and copies `RECORD.FIELD`; Escape exits; Alt+Shift+C copies again.
- **Page Info:** Richer than Menu/Component — ToolsRel, DB, portal/node/site, UI mode; Markdown copy for defects.
- **New Window:** Open the current component in a `_newwin` style window/tab.
- **Backup:** Export full settings JSON from Favorites tab tools.
