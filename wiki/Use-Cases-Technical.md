# Use cases — Technical

- **Trace:** Configure PeopleCode/SQL flags (or named presets: Off / Default / SQL / PeopleCode / Verbose) in Options → Tracing, then toggle Trace on the bar. Requires security access to delivered trace components. Cross-tab sync updates other tabs. 🔒 means UTILITIES access is missing.
- **Field Inspector:** Orange icons beside fields on Classic and Fluid (`ps_box-*`); Rec/Fld/Row (+ Label / Work badge); click locks and copies `RECORD.FIELD`; Escape exits; Alt+Shift+C copies again.
- **Page Info:** Richer than Menu/Component — ToolsRel, DB, portal/node/site, UI mode; Markdown copy for defects.
- **Go to / New Window:** Jump to Menu.Component.Market (Alt+Shift+G) or open the current component in a `_newwin` tab.
- **Backup:** Export full settings JSON from Favorites tab tools.
